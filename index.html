The Developer

I can tell you that I've been involved professionally in web development for over 10 years, but technology and ideas change so rapidly -- What’s been popular 10 years ago has little significance today. My philosophy is simple: develop software for end-users, utilize modern technology, build on experience and don't let the past dictate the future. But, I'm getting ahead of myself...

The Technology

... there is a difference between what's in vogue and what's practical. Technology is subjective. As a developer, my goal is to be stack-agnostic as much as possible. My current preference is JavaScript, in particular Node.js, Angular.js, Express.js, but I've developed on ASP.NET C# stack, PHP and Laravel, MySQL, MSSQL, Mongodb as well jQuery. Indeed, let's not forget the essentials HTML5 and CSS3.

The Designer

You can take the best technology and development techniques, but without a great user experience, the app will serve little value. Designing a great UX plays a pivotal part in my everyday process. It’s not just creating a great looking UI -- it’s about using data points, interviews, prototypes and personas. By understanding the user is where we create value and an awesome web application.
